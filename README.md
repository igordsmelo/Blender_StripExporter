# Blender_StripExporter
Blender add-on to export individual strips from your video edits.
**Tested on W11 running blender 3.4**


![img.png](img.png)

# Install:
1. download .py file
2. open blender
3. edit>preferences>add-ons>install
4. selected .py file.
5. toggle it on.
